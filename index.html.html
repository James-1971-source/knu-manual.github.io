소개

KNU-VoiceAnouncer는 경북대학교 생활관을 위해 개발된 다국어 안내방송 프로그램입니다. 이 프로그램을 통해 한국어, 영어, 일본어, 중국어, 인도어로 안내방송을 제작하고 송출할 수 있습니다. 또한 방송 예약 기능을 통해 지정된 시간에 자동으로 방송을 송출할 수 있습니다.

KNU-VoiceAnouncer 프로그램 메인 화면

프로그램의 주요 기능은 다음과 같습니다:

  • 다국어 텍스트 번역 (한국어, 영어, 일본어, 중국어, 인도어)
  • 텍스트를 음성으로 변환하는 TTS(Text-to-Speech) 기능
  • 생성된 음성 파일의 미리듣기 및 방송 실행
  • 날짜와 시간을 지정한 예약 방송 설정
  • 미리듣기 및 방송실행을 위한 별도의 스피커 설정
  • 효과음 및 음성 파일의 볼륨 독립 제어

프로그램 시작하기

프로그램을 실행하면 메인 화면이 표시됩니다. 화면 상단에는 경북대학교 생활관 로고와, 프로그램 제목, 현재 날짜와 시간이 표시됩니다.

메인 화면은 다음 세 개의 탭으로 구성되어 있습니다:

일반 방송 설정
예약 방송 설정
환경 설정
  • 일반 방송 설정: 텍스트 입력, 번역, 음성 변환 및 방송 제어
  • 예약 방송 설정: 미리 제작된 방송의 예약 시간 설정
  • 환경 설정: 스피커 설정 및 시스템 오디오 설정

일반 방송 설정

일반 방송 설정 탭에서는 방송 내용을 작성하고 번역하여 음성으로 변환할 수 있습니다.

텍스트 입력 및 변환

텍스트 입력

텍스트 입력 영역에 안내방송 내용을 입력합니다.

텍스트 파일에서 내용을 불러오려면 하단의 '텍스트 파일 열기' 버튼을 클릭합니다.

작성한 내용을 파일로 저장하려면 '텍스트 저장' 버튼을 클릭합니다.

번역

번역할 언어를 드롭다운 메뉴에서 선택합니다 (한국어, 영어, 일본어, 중국어, 인도어).

'번역' 버튼을 클릭하면 입력된 텍스트가 선택한 언어로 번역됩니다.

번역할 언어: 한국어 ▼

음성 변환

텍스트 입력 후 '음성 변환' 버튼을 클릭하면 텍스트가 선택한 언어의 음성 파일로 변환됩니다.

변환된 음성 파일은 자동으로 파일 목록에 추가됩니다.

🔊 음성 변환

볼륨 조절

'음성파일 볼륨' 슬라이더로 음성 방송의 볼륨을 조절합니다.

'효과음 볼륨' 슬라이더로 효과음의 볼륨을 조절합니다.

⚠️ 주의: 음성 파일은 '음성파일 볼륨'에만 영향을 받고, 효과음은 '효과음 볼륨'에만 영향을 받습니다.

예약 추가

예약 추가 절차:

  1. '일반 방송 설정' 탭에서 원하는 음성 파일을 선택합니다.
  2. '예약 방송 설정' 탭으로 이동합니다.
  3. 예약 날짜와 시간을 설정합니다.
  4. '예약 추가' 버튼을 클릭하여 예약을 등록합니다.

예약 실행

  • 예약된 시간이 되면 자동으로 해당 방송이 실행됩니다.
  • 예약 방송이 실행되면 알림창이 표시되고, 실행된 예약은 목록에서 자동으로 삭제됩니다.
  • ⚠️ 주의: 예약 방송 실행을 위해서는 예약 시간에 프로그램이 실행 중이어야 합니다.

문제 해결

자주 발생하는 문제

스피커 설정 문제

스피커 설정이 적용되지 않는 경우:

  • '시스템 오디오 설정 열기' 버튼을 클릭하여 Windows 오디오 설정에서 기본 출력 장치를 변경하세요.
  • 스피커가 올바르게 연결되어 있는지 확인하세요.

파일 관리 문제

파일 삭제가 되지 않는 경우:

  • 모든 방송 및 미리듣기를 중지하세요.
  • 잠시 후 다시 시도하세요.
  • 프로그램을 재시작한 후 다시 시도하세요.

번역 및 음성 변환 문제

번역 또는 음성 변환 오류 발생 시:

  • 인터넷 연결을 확인하세요.
  • 너무 긴 텍스트는 여러 부분으로 나누어 변환하세요.

예약 방송 문제

예약 방송이 실행되지 않는 경우:

  • 예약 시간이 현재 시간 이후로 설정되었는지 확인하세요.
  • 방송에 사용될 파일이 삭제되지 않았는지 확인하세요.
  • 예약한 시간에 프로그램이 실행 중인지 확인하세요.

기타 문제해결 팁

  • 프로그램을 재시작하면 대부분의 일시적인 문제가 해결됩니다.
  • 음성 파일과 효과음 파일은 별도의 볼륨 슬라이더로 조절됩니다. 적절한 볼륨 슬라이더를 사용하고 있는지 확인하세요.
  • 스피커 설정 후 테스트 기능을 사용하여 올바르게 설정되었는지 확인하세요.
